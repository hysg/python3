# python封包与解包
## 解包
解包把集合里边元素复制给变量
定义两个变量a和b
```
>>>a, b = 1, 2
```
交换两个变量的值
```
a, b = 2, 1
```
打印两个变量的值
```
>>>a
2
>>>b
1
```
可以在结构稍微复杂情况解包
```
>>>e, (f, g), h = [1, [2,3],4]
```
然后打印变量的值
```
>>>e
1
>>>f
2
>>>g
3
>>>h
4
```
解包，也可以掐头去尾，只关注你需要的变量,用``*_``表示准备舍弃的变量
```
>>>c = list(range(10))
>>>head, *_, tail = c
```
打印head和tail
```
>>>head
0
>>>tail
9
```
## 封包
封包是把变量构建成元组
```
>>>t = 1, 2, 3
```
打印变量t
```
>>>t
(1, 2, 3)
```
判断变量t的类型
```
>>>type(t)
<class 'tuple'>
```
