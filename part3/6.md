## python bytes
bytes是python 3特有的数据类型，python 2里边不区分bytes和str。
### bytes和str区别
bytes和str区别就是bytes是byte的序列，而str是unicode的序列。
例如
```
>>>s = '回忆时光'
>>>b = s.encode()
```
打印b
```
>>>print(b)
b'\xe5\x9b\x9e\xe5\xbf\x86\xe6\x97\xb6\xe5\x85\x89'
```
遍历字符串
```
fot i in s:
    print(s)
```
打印结果是：
```
回
忆
时
光
```
遍历bytes
```
fot x in b
    print(x)
```
打印结果是
```
229
155
158
229
191
134
230
151
182
229
133
137
```
*str使用encode方法转换为bytes，bytes通过decode方法转化为str*
