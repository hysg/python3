# python变量
## 变量含义
变量是一个指针，指向一块内存。python中的变量不仅可以是数字，还能是任意数据类型。
## 变量名命名规则
+ 变量名必须以数字、字母或者下划线开头
+ 变量名只能包含数字、字母或者下划线
+ 不能是python解释器的保留字

## 变量赋值
python中的变量赋值，不用事先指定其变量类型。
```
a = 1
b = 1.2
c = 'hello'
d = True
# a 是整数
print(a)
# b 是浮点数
print(b)
# c 是字符型
print(c)
# d 是布尔型
print(d)
```
运行结果是
```
a = 1
b = 1.2
c = 'hello'
d = True
```
