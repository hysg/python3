# python数据类型
Python中有六大数据类型，分别是:
+ Number(数值型)
+ String(字符型)
+ List(列表)
+ Tuple(元组)
+ Sets(集合)
+ Dictionary(字典)

## Number
### int
长整形，比如
```
a = 123
```
### float
浮点数,比如
```
b = 1.23
```
### bool
布尔型，比如
```
c = True
d = False
```
### complex
复数,比如
```
e = 3.14j
```

### 判断字符类型
可以使用`type`来判断字符类型，在终端输入`python`命令进入python的交互界面
```
type(a)
# a 是长整形
<class 'int'>
type(b)
# b 是浮点型
<class 'float'>
# c 和 d 是布尔型
type(c)
<class 'bool'>
type(d)
<class 'bool'>
```
